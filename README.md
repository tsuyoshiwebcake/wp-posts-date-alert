# WP Posts Date Alert
WP Posts Date Alertは投稿の公開日時と現在の日時を比較して指定した期間より古い投稿の場合、メッセージを表示するWordPressプラグインです。

## License
GPL License

## Introduction
管理者でログインしてメニューの **設定** から **WP Posts Date Alert** を選択します。

設定を入力して、 **変更を保存** を押してください。
